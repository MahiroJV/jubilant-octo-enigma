## 🚀 Getting Started
 
### Requirements
- 🐧 Any Linux distro (Arch preferred 😎)
- 🖥️ Terminal (the real IDE)
- ☕ A hot drink and curiosity
 
### Clone The Repo
```bash
git clone https://github.com/yourusername/your-repo-name.git
cd your-repo-name
```
 
### Run A Shell Script
```bash
chmod +x script.sh
./script.sh
```
